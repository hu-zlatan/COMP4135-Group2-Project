\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
Tactical Cards is a Unity-based tactical card combat prototype designed for teenagers and casual strategy players who benefit from readable, small-scope decision making. The delivered project provides a complete single-battle loop with a title screen, a playable grid encounter, a result screen, deterministic enemy response, and a themed runtime HUD. The design focuses on one compact tactical scenario rather than a larger campaign structure so that interaction order, board state, card effects, and player feedback remain easy to understand. This report concentrates on the system itself: the requirements that shape the battle loop, the architecture used to implement it, the runtime interaction model, the user interface design, the current validation results, and the main technical limitations and extension paths.
